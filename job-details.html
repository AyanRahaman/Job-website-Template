Full Stack MERN Stack Developer

Noida Full Time

Job description

We are seeking an experienced Full Stack MERN Stack Developer to join our development team and work on building scalable web applications. The ideal candidate will have expertise in the MERN stack (MongoDB, Express.js, React, and Node.js), a passion for clean, maintainable code, and a desire to deliver high-quality, user-centric solutions. This role involves both frontend and backend development, working closely with cross-functional teams to create dynamic, responsive, and efficient web applications.

Job description

  • Develop Full-Stack Web Applications – Design and develop full-stack applications using MongoDB, Express.js, React, and Node.js.
  • Frontend Development – Build responsive, user-friendly interfaces using React, ensuring seamless user experiences across devices.
  • Backend Development – Develop robust server-side logic, APIs, and databases using Node.js and Express.js.
  • Database Management – Design and manage NoSQL databases using MongoDB, optimizing queries and ensuring data integrity.
  • API Development and Integration – Create and maintain RESTful APIs to connect the application with other services and third-party systems.
  • Code Optimization and Debugging – Identify, debug, and resolve performance issues, writing clean, efficient, and maintainable code.
  • Collaborate with Cross-Functional Teams – Work with UI/UX designers, project managers, and other developers to deliver a cohesive application.
  • Maintain Code Quality – Ensure high standards in code quality, testing, and adherence to best practices.
  • Security and Data Protection – Implement security protocols and data protection measures in applications to protect against vulnerabilities.
  • Stay Updated with Latest Technologies – Continuously learn and apply emerging trends and tools in the MERN stack ecosystem to improve application performance and user experience.

Job description

  • Education – Bachelor’s degree in Computer Science, Software Engineering, or a related field (or equivalent experience).
  • Experience – Proven experience as a Full Stack Developer with expertise in the MERN stack; typically 2+ years.
  • Technical Skills:
    • Frontend – Strong proficiency in React, HTML, CSS, and JavaScript (ES6+).
    • Backend – Expertise in Node.js and Express.js for server-side programming.
    • Database – Experience with MongoDB, including database design, query optimization, and performance tuning.
    • Version Control – Proficient in Git and familiar with collaborative workflows.
  • Knowledge of RESTful APIs and WebSockets – Ability to design and consume RESTful APIs and work with real-time data where necessary.
  • Problem-Solving Skills – Strong analytical and problem-solving skills, with attention to detail. Agile Development – Familiarity with Agile methodologies and experience working in an Agile team.
  • Team Collaboration – Strong communication skills and experience working effectively with other developers, designers, and stakeholders.
  • Adaptability – Ability to learn and adapt to new technologies and a fast-changing environment.

Benefits

  • Competitive salary and performance-based incentives.
  • Health, dental, and vision insurance for employees and dependents.
  • Paid time off, including vacation, sick leave, and holidays.
  • Opportunities for professional development, training, and certifications.
  • Flexible work arrangements and remote work options.
  • Company-sponsored team events and regular team-building activities.
This is an excellent opportunity for a Full Stack MERN Stack Developer to work on impactful projects and develop innovative web applications as part of a collaborative and forward-thinking team. Join us and contribute to building solutions that drive our company’s success.
